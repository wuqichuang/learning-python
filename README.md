## 🗺️ 学习路线

- **Day 1–5 基础阶段**：每天「讲解 1 小时 → 自己练 2 小时 → 答疑」，MySQL 建表/插入/查询在这几天必须掌握
- **Day 6–15 项目阶段**：用前 5 天的全部知识开发学生管理系统（MySQL 存储 + Python 操作 + 服务器部署）
- **Day 16–20 进阶阶段**：子查询/视图、正则清洗、窗口函数、Flask 网页、自动备份
